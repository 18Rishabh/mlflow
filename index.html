MLflow (currently in alpha) is an open source platform to manage the ML lifecycle, including experimentation, reproducibility and deployment. It currently offers three components:

  • MLflow Tracking

    Record and query experiments: code, data, config, and results.

    Read more

  • MLflow Projects

    Packaging format for reproducible runs on any platform.

    Read more

  • MLflow Models

    General format for sending models to diverse deployment tools.
