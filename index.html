MLflow

An open source platform for managing the end-to-end machine learning lifecycle.

  • Works with any ML library, language & existing code
  • Runs the same way on any cloud
  • Designed for 1 or 100,000 person orgs
  • Scales to big data with Apache Spark

Reproducible runs using any ML library or framework

Run the same way on your laptop or in the cloud

Record Experiment runs to track & query your results

See the parameters, results, code, data from each run

Easy model deployment and management

Download or deploy any saved model to various platforms

Download and get started.

Please clone mlflow from github at github.com/databricks/mlflow
